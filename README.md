# RepositórioV2

Este repositório contém um projeto web simples com página(s) estática(s) em HTML, estilos em CSS e comportamento em JavaScript.

## Linguagens utilizadas

- HTML
- CSS
- JavaScript

> O projeto também possui um arquivo `package.json` com dependências de Node.js (`animejs` e `express`)

## Estrutura de arquivos

```
├── 📁 CSS
│   └── 🎨 style.css
├── 📁 HTML
│   └── 🌐 index.html
├── 📁 IMG
│   ├── 🖼️ bg-image-dark.jpg
│   ├── 🖼️ bg-image-light.jpg
│   ├── 🖼️ letra-d.png
│   └── 🖼️ thiago.JPEG
├── 📁 JAVASCRIPT
│   └── 📄 script.js
├── 📝 README.md
├── ⚙️ package-lock.json
└── ⚙️ package.json
```

## Descrição dos arquivos

- `HTML/index.html`: página principal do projeto.
- `HTML/contato.html`: página de contato.
- `CSS/style.css`: estilos do site.
- `JAVASCRIPT/script.js`: scripts JavaScript para interatividade.
- `IMG/`: pasta para imagens do projeto.
- `package.json`: arquivo de configuração de pacotes do Node.js.
- `package-lock.json`: bloqueio exato de versões das dependências.

## Como usar

1. Abra `HTML/index.html` em um navegador para visualizar o site.
